Все задачи импорта данных, которые wh_*, можно запустить из командной строки.
Для этого нужно обязательно сменить директорию на ./tasks и запустить

 \>node $launch <имя файла wh_*> "{}"

Последний параметр "{}" - это json для передачи всех параметров. 
Например если в задаче используется params.date, то можно передать
дату так: "{\"date\":\"2011-01-01\"}" (внутренние кавычки экранировать через слэш \)
